# Update Packages

## NetBSD

```
pkgin upgrade
```

## FreeBSD

```
pkg upgrade
```

## OpenBSD

```
pkg_add -uvi
```

https://linux-audit.com/updating-all-openbsd-packages-with-pkg_add/

## Arch

```
pacman -Syu
```

## Fedora

```
dnf upgrade
```

## Gentoo

```
emerge --sync
```

## Alpine

```
apk upgrade
```
